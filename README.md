## 📌 Overview

**Asfor (عصفور)** is an advanced, end-to-end intelligent platform designed to combat the spread of misinformation and fake news. Inspired by the concept of a swift bird spreading validated truths, **Asfor** acts as a digital verification assistant. 

By leveraging cutting-edge Natural Language Processing (NLP), localized machine learning models, and real-time live web verification, the system analyzes text, cross-references statements, and delivers a transparent credibility score to help users distinguish facts from fabrication in seconds.

---

## 🚀 Key Features

*   **Intelligent Text Classification:** Utilizes dedicated Machine Learning models optimized for text pre-processing (tokenization, stemming) and classification.
*   **Real-time Fact Checking:** Integrates dynamic search APIs to scan the live web and verify active headlines or claims instantly.
*   **Advanced LLM Reasoning:** Powered by state-of-the-art Large Language Models via cloud inference to provide contextual analysis and detailed explanations for the verification results.
*   **Sleek & Interactive UI:** A modern, highly responsive dashboard tailored for seamless user interaction and clear data visualization.
